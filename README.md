# IDX Property Search
A production-style, full-stack real estate property search platform built as part of the IDX Exchange SDE Internship Program. This project simulates a Zillow / Redfin-like application and emphasizes backend API design, database modeling, frontend integration, and real-world software engineering practices.

# Tech Stacks

# Project Goals
Build a Zillow-style property search platform
Design and implement RESTful backend APIs
Model and query relational property data efficiently
Integrate frontend UI with backend services
Practice real-world Git workflows and code reviews
Develop full-stack ownership from database to UI
Each intern owns their own end-to-end implementation, while collaborating with pod members through weekly reviews and discussions.
# Core Features
Property search with filters (price, beds, baths, city)
Pagination and sorting
Property detail page
Open house schedule integration
Backend input validation and error handling
Indexed SQL queries for performance
Responsive frontend UI
